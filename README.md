
LineBot 發票對獎小精靈

>>簡介<<
透過Python網路爬蟲財政部CSS資料，結合Line messaging API實做發票兌獎機器人

>>使用說明<<
1.掃描QRcode將Linebot 發票對獎小精靈加入好友

![image](https://github.com/ETATEK/Linebot_Invoice/blob/main/QRcode.png)

2.選單輸入相關指令

├── @發票後三碼對獎           
├── @前期中獎號碼              
├── @本期中獎號碼

3. APP 畫面截圖
